# ピンボール
スペースキー長押しでためる
離したらボール発射
Vキーで左パドル操作
Bキーで右パドル操作

制作時間　
5時間

ルール　
ボールを跳ね返してスコアを稼ぐ

# 紙相撲
矢印キーでたたく方向を操作
スペースでたたく

制作時間　
５時間

ルール　
倒れない程度にたたきながら相手を倒す

# 弾幕ゲーム
矢印キーで操作

制作時間　
20時間

ルール　
敵からの弾幕を避けてHPを削り切ったら勝利

ハードモード
当たり判定大
敵HP二倍
音量大
